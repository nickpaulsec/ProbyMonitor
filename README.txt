ProbyMonitor 

Platform : Unix (Tested on Parrot OS and Kali)

ProbyMonitor is a data exfiltration detection tool.
It aims to assist in detection of data exfiltration through 802.11 probe requests.

REQUIREMENTS
-------------------------------------------------------------------
	External Imports Required All Avaliable Through PyPi
import scapy
import wifi


	Misc.
- A wireless interface that supports monitor mode
- Root when running the script
